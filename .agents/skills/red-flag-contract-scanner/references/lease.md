# 租約專屬紅旗清單(Lease / Rental Agreement)

以台灣租賃住宅為主要情境(《租賃住宅市場發展及管理條例》、內政部「住宅租賃定型化契約應記載及不得記載事項」)。商辦租賃稍有不同,看到商辦合約時要主動提醒「我用住宅租賃標準審,商辦慣例不同,某些條款可能無法直接套用」。

## 必看的台灣法定保護(房客的盾牌)

先檢查這些「房客本來就有的權利」是否被合約剝奪 —— 一旦合約寫了「房客拋棄 / 同意 / 視為放棄」,就要警示這條可能違法無效:

1. **押金不得超過 2 個月租金**(土地法第 99 條)
2. **房東不得拒絕房客遷入登記**(內政部不得記載事項)
3. **稅費由房客負擔的約定無效**(房屋稅、地價稅、所得稅原則上是房東責任)
4. **裝修費押金不得單獨設立**(實務常見「裝潢押金 5 萬」→ 違法)
5. **修繕義務原則由房東承擔**(民法第 423 條)
6. **租金調漲須經合意**(契約期間內)
7. **提前解約權**:房東若要終止 → 限定情境(積欠 2 月租金、轉租、違約使用);房客若要終止 → 至少在「房屋有重大瑕疵房東不修」時可主張

## 房東常偷塞的紅旗條款

### 🚨 紅旗 #1:「不得遷入戶籍 / 不得辦理租屋補助 / 不得申報房屋租金支出」

**為何紅旗**:房東通常是想逃稅。對你而言:
- 不能報所得稅扣除(每年最高 18 萬)
- 不能申請租金補貼
- 不能設籍 → 影響投票、子女就學、政府服務
- **這條在台灣是法定不得記載事項**,寫了無效,但房東會用「你違約我就終止」威脅

**協商建議**:直接刪除。房東怕的是稅,可以協商「我報稅 → 但房租提高 X 元 cover 房東稅務」也可以,但不能剝奪你的權利。

### 🚨 紅旗 #2:押金額度過高 / 押金扣抵範圍過寬

**長什麼樣**:
- 押金 3 個月以上(法定上限 2 個月,超過部分可請求返還)
- 「押金可抵充任何違約金、修繕費、清潔費、垃圾清運費」
- 「退租時房東有權於 [60/90/180 天] 內返還押金」(實務應於交屋驗收後合理期間,通常 1-2 週)
- 「押金不得抵充最後一期租金」(法律上其實可以,但房東不想)

### 🚨 紅旗 #3:修繕責任倒置

**長什麼樣**:
- 「房屋一切修繕由房客自費」
- 「設備自然耗損之維修由房客負擔」
- 「水電管線、馬桶、熱水器、漏水修繕由房客處理」

**法律上的常識**:
- 自然耗損 / 結構性問題 / 既有設備故障 → 房東責任
- 房客使用不當造成的損壞 → 房客責任
- 消耗品(燈泡、濾網)→ 房客責任(慣例)

寫成「全部房客負擔」就是把房東責任轉嫁,是紅旗。

### 🚨 紅旗 #4:提前解約懲罰過重

**長什麼樣**:
- 「房客提前解約須賠償剩餘月份全額租金」(等於沒解約)
- 「提前解約沒收押金 + 賠償 X 個月租金」(雙重懲罰)
- 房東提前終止 → 賠 1 個月;房客提前終止 → 賠 3-6 個月(嚴重不對等)

**正常範圍**:房客提前解約賠 1-2 個月租金 + 提前 1 個月通知,業界標準。超過就紅旗。

### 🚨 紅旗 #5:單方漲租權

**長什麼樣**:
- 「房東得依物價指數調整租金」
- 「合約期間每年調漲 [X%]」 (要看比例是否合理 + 是否合意)
- 「續約時房東得單方決定新租金」

**檢查**:契約期間內漲租 → 法律上需合意,單方漲無效;但寫了仍會造成爭議。續約時是另一回事。

### 🚨 紅旗 #6:房東隨時進屋權

**長什麼樣**:
- 「房東得隨時進入查看房屋狀況」
- 「房東持有備用鑰匙得於必要時進入」

**正常範圍**:應約定「需提前 24-48 小時通知 + 房客同意 + 緊急狀況除外」。不寫就是紅旗。

### 🚨 紅旗 #7:擔保人 / 連帶保證人條款過嚴

**長什麼樣**:
- 連帶保證人責任無上限、無時限
- 連帶保證人需負擔「所有違約責任、損害賠償、訴訟費」
- 沒有保證人退場機制

**檢查**:擔保責任應有上限(通常 12 個月租金為合理),且應隨契約終止而終止。

### 🚨 紅旗 #8:轉租 / 同居人限制過嚴

**長什麼樣**:
- 「不得有任何訪客留宿」
- 「同居人需經房東書面同意 + 額外加收費用」
- 「禁止任何形式之短期轉租」(這部分若指 Airbnb 式商業轉租是合理的,但若連讓朋友來住一晚都禁止 → 過嚴)

### ⚠️ 注意 #1:設備清單與點交

**檢查**:
- 是否有完整設備清單?(冷氣、熱水器、家具一一列出)
- 既有瑕疵是否書面記錄?(漆面剝落、地板刮痕、家具狀況)
- 沒寫清楚 → 退租時房東可宣稱「這是你弄壞的」要求賠償

**提醒使用者**:點交時拍照 + 錄影 + 雙方簽名確認清單。

### ⚠️ 注意 #2:水電費 / 管理費 / 網路費分擔

**檢查**:
- 水電費誰繳?是否依實際度數?還是房東「估收」(套房常見,但要確認費率是否合理 —— 不能比台電 4 倍)
- 管理費含在租金內還是另計?
- 網路費 / 第四台是否強制 + 費用合理?

### ⚠️ 注意 #3:寵物條款

**檢查**:
- 完全禁止 → 簽約前想清楚未來會不會養
- 有條件允許 → 額外押金合理嗎?

### ⚠️ 注意 #4:違約使用條款

**長什麼樣**:「不得作為營業 / 商業 / 工作室用途」

**檢查**:你會在家工作嗎?接案?直播?在家工作通常 OK 但若有實體客戶上門 = 商業使用,可能被房東主張違約。

### ✅ 通常正常的條款

- 維持房屋原狀(不得擅自裝修)
- 不得從事違法行為
- 房屋毀損時房客通知房東義務
- 退租時須回復原狀(在合理使用範圍內)
- 房客需投保火險或第三人責任險(若費用合理)

## 簽租約前的實務檢查清單

幫使用者列出來:

1. **房東身分**:核對房屋謄本(地政司 EChannel 可查),確認對方是登記所有人或有合法授權
2. **建物用途**:謄本上的「主要用途」是住宅還是商辦?工業用地當住宅 → 違建風險
3. **押金/租金匯款帳戶**:應為房東本人帳戶
4. **完整設備清單拍照存證**
5. **租賃契約是否報請公證 / 上傳「租賃住宅服務商業同業公會」備查**(有公證才能逕付強制執行)
6. **房屋是否有抵押權設定**(會不會被法拍 → 你要搬)
7. **是否申請租金補貼**(年輕人 / 育兒族常見)→ 確認房東不反對

## 房客可協商的條款

不要讓使用者以為合約都不能改。實務上可以協商的:

- 押金 → 業界標準是 2 個月,可談 1 個月(尤其房市買方市場時)
- 通知遷出期 → 提前 30-60 天合理,可協商
- 修繕分工 → 大型修繕房東負擔、小型由房客
- 違約金 → 提前解約賠 1 個月租金為合理範圍
- 設備添購 / 油漆 → 簽約前談,房東通常願意小額讓步
- 冷氣 / 熱水器老舊 → 要求簽約前更換或承諾故障即修

告訴使用者:**簽下去之前你最有議價權,簽下去之後就是房東的天下**。
