# 會員 / 訂閱服務合約紅旗清單(Membership & Subscription)

涵蓋:健身房、瑜伽教室、補習班、線上課程、SaaS 訂閱、串流訂閱、網購會員、美容 / 醫美課程、雲端儲存、會員制商店等。

健身房是最經典的「進場容易出場難」合約,以下原則同樣適用於其他訂閱服務。

## 必看的台灣消保法保護

先檢查這些法定權利是否被剝奪:

1. **7 天猶豫期**(消費者保護法第 19 條,但僅限「通訊交易」與部分特定交易,健身房現場簽約不適用)
2. **健身房定型化契約應記載及不得記載事項**(行政院消費者保護處 2018 公告):
   - 7 日內可無條件解約退費
   - 不得約定終身會員 / 不得約定不可解約
   - 解約退款最遲 30 日內
   - 違約金不得超過剩餘期間月費 20%
   - 教練課與會員費應分別計價、分別退款
3. **預收款保護**:健身房收取超過 1 個月費用,法定要有「履約保證」(由銀行 / 保險公司擔保)

## 進場最常見的紅旗條款

### 🚨 紅旗 #1:綁約期間過長 + 違約金高

**長什麼樣**:
- 「會員須履約 24 / 36 個月,提前解約須賠償剩餘月費 50% / 80% / 100%」
- 「教練課程須於 X 個月內使用完畢,逾期不退費」
- 「年費一次付清優惠價,中途解約恕不退費」

**法定上限**:健身房違約金 ≤ 剩餘月費 20%。**寫超過 = 違法,但他們還是會寫**。

**檢查重點**:把「總費用 ÷ 綁約月份」算出每月實際成本,跟廣告月費對照。

### 🚨 紅旗 #2:退會通知期長 + 通知方式刁難

**長什麼樣**:
- 「會員需於到期前 60 / 90 天以掛號信通知總公司」
- 「退會須親自至 [指定門市] 辦理,且僅週一至週五 10:00-17:00」
- 「電話、email 通知不生效力」
- 「退會表單須會員本人到場填寫並經店長 / 店經理簽核」

**為何紅旗**:把退會做成「比加入難 10 倍」是設計性的。常見話術「我們系統還沒看到你的通知」「主管不在無法簽」「需要等下個月才能處理」。

**理想條款**:電子郵件 / 簡訊 / 線上申請即可,30 天通知期。

### 🚨 紅旗 #3:自動續約陷阱

**長什麼樣**:
- 「合約到期後若未通知終止,自動續約 [12 / 24] 個月」
- 「續約後仍適用原合約所有條款」(包括違約金 → 你被綁第二次)
- 「續約價格依當時公告價」(可能漲了)

**在台灣的法律保護**:健身房定型化契約規定「自動續約前應於 60 日前以書面 / 電子方式通知會員」並提供拒絕續約管道。沒做到 → 自動續約無效。

### 🚨 紅旗 #4:教練課 / 點數不退費

**長什麼樣**:
- 「教練課購買後恕不退費 / 不可轉讓 / 不可延期」
- 「未於有效期內使用之點數視同放棄」
- 「教練離職 / 轉店,會員不得以此為由解約退費」

**法律上**:健身房定型化契約規定教練課應可比例退費(扣掉已使用部分 + 合理手續費)。終身有效或長效是業者承諾,寫「過期作廢」是紅旗。

### 🚨 紅旗 #5:單方漲價 / 變更服務條款

**長什麼樣**:
- 「業者得依市場狀況調整月費」
- 「設施使用範圍、營業時間、課程內容得由業者隨時變更」
- 「分店關閉、合併由業者全權決定,會員不得異議」

**檢查**:有沒有「漲價 → 你可以無條件解約退剩餘費用」?沒有就紅旗。

### 🚨 紅旗 #6:身體狀況免責 + 受傷免責

**長什麼樣**:
- 「會員於健身房內發生之任何意外、傷害、疾病,業者概不負責」
- 「會員應自行評估身體狀況,因運動產生之健康問題與業者無關」

**法律上**:業者對「設施安全、教練專業、合理督導義務」仍須負責。完全免責條款違反消保法,通常無效,但他們還是寫,出事時會用這條打官司。

### 🚨 紅旗 #7:會籍轉讓 / 暫停費用刁難

**長什麼樣**:
- 「會籍轉讓需收取手續費 [3000-5000 元]」
- 「會籍暫停每月仍收取維持費」
- 「暫停最多 X 個月,逾期會籍恢復計費」
- 「暫停僅限疾病、海外工作,需附證明」

**檢查**:遇到出國、生病、懷孕、搬家時你能怎麼辦?彈性越低 → 風險越高。

### 🚨 紅旗 #8:預收款無履約保證

**長什麼樣**:你付了 30,000 元年費,但合約裡找不到「銀行履約保證」「信託專戶」字樣。

**為何紅旗**:健身房倒閉 → 你的錢拿不回來。台灣每隔幾年就有大型健身房倒閉(World Gym 早期、各地小型業者),沒有履約保證的預付款 = 高風險。

**檢查**:法定要求超過 1 個月費用須有履約保證。沒寫 = 違法或業者沒做。

### ⚠️ 注意 #1:加值服務綁約

**長什麼樣**:你以為簽的是會員費,結果被附加教練課程包、營養品、量身打造方案、儀器療程,綁進總額。

**檢查**:把所有「服務項目」分項列出,看有沒有被打包進來自己沒注意。

### ⚠️ 注意 #2:設施使用限制

**檢查**:
- 高峰時段(晚上 6-9 點)是否限制使用部分設施
- 跨店使用是否需額外付費
- 課程預約是否需另外付費 / 提前多久搶

### ⚠️ 注意 #3:個資 / 影像授權

**長什麼樣**:「會員同意業者使用會員肖像於宣傳用途」

**檢查**:你 OK 嗎?能不能拒絕?拒絕會不會被刁難?

### ✅ 通常正常的條款

- 進場需出示會員卡 / 身分證明
- 不得使用他人會員身分
- 應遵守場館秩序、衣著規定
- 業者保留拒絕特定行為(吸毒、酒醉、騷擾)會員入場之權利

## SaaS / 線上訂閱服務的特殊紅旗

如果是線上服務,額外注意:

### 🚨 自動扣款難取消

- 「取消訂閱需透過 [特定隱藏路徑]」
- 「次月扣款後恕不退費」
- 「降級方案後當月仍按原方案收費」

### 🚨 資料 hostage 情況

- 「降級 / 取消後 [X 天] 內可下載資料,逾期刪除」
- 「降級後只能看資料,不能匯出」
- 「企業帳戶資料屬於業者,離職員工不得攜帶離開」

### 🚨 用量計費突發

- 「超過免費額度後依用量計費,單位 [模糊]」
- 「業者得依需要調整免費額度」(下個月你的免費額度被調低 → 自動扣費)

### 🚨 地區 / 司法管轄

- 美國公司 SaaS 寫「適用美國加州法、爭議由舊金山法院管轄」 → 你打不起官司

## 簽會員合約前的實務檢查清單

1. **算總價**:綁約期 × 月費 + 入會費 + 教練課 + 加購 = 你實際要付的
2. **試算違約成本**:現在解約要賠多少?中途解約要賠多少?
3. **查業者財務狀況**:Google 業者名稱 + 「倒閉 / 跑路 / 結束營業」
4. **查履約保證**:合約有沒有寫銀行 / 信託?
5. **業務口頭承諾**:「之後可以暫停」「不喜歡可以退」「教練離職可換」全部要求白紙黑字寫進合約
6. **退會程序**:現在就試問「萬一我下個月想退,該怎麼辦?」要求對方寫成書面說明
7. **消保專線**:1950 / 各縣市消保官 → 萬一發生爭議的求助管道

## 已經簽了想脫身怎麼辦(使用者常追問)

如果使用者問「我已經簽了該怎麼辦」:

1. **冷靜期**(7 天內):不管合約怎麼寫,法定 7 天可無條件解約(健身房業)
2. **業者違約**:漲價、減服務、設備不堪使用、教練品質明顯下降 → 主張對方違約終止
3. **重大情事變更**:出國、重病、懷孕、搬家(距離過遠)→ 主張情事變更
4. **違反消保法 / 不得記載事項**:那條條款本來就無效,可拒絕履行
5. **直接向消保官申訴**:1950 / 各縣市消保專線
6. **訴諸新聞 / 社群**:不是法律手段,但實務上有效

提醒:**不要直接停止扣款**(可能被告違約),要先「書面通知終止 + 主張理由 + 拒繳後續」,留下完整紀錄。
