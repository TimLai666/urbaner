# Pressure Scenario 02: 地方品牌限時活動

## Purpose

驗證 skill 是否能在台灣地方品牌情境下，正確使用錨定與框架，而不是寫成空泛文青文案。

## Scenario

使用者要為地方品牌做週末限定快閃活動。

已知資料：

- 產品：阿里山高山茶體驗組
- 受眾：喜歡送禮與在地選品的 30-50 歲消費者
- 目標：提高兩日快閃活動的來店與下單
- channel：event
- conversion_stage：hot
- urgency_context：只做本週末兩天，現場才有組合價
- proof_assets：有現場活動時間、原價與套組價

## Baseline Without Skill

未載入 skill 時，模型常見缺點：

- 容易只強調在地、溫度、支持小農，缺少成交推力
- 會提限時，但不會把價格比較設計成錨定
- 台灣語境不穩，容易出現不自然的泛中文案

## Pass Criteria

- 至少用到 `嚇唬 / 失落驅動`、`錨定`、`框架` 中的兩項
- 有清楚說明為何這樣搭配
- 文案與策略用繁體中文、台灣用語
- 有將「原價 / 活動價 / 組合價」轉成訊息優勢
- 有在 `risk_flags` 中提醒限時與價差都需真實

## Fail Signs

- 只寫品牌故事，沒有轉換動作
- 沒有價格比較或限時條件設計
- 沒有 `risk_flags`
- 使用不自然或非台灣語感措辭
