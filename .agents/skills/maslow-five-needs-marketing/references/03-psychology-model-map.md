# Psychology Model Map

以 IQ 228 顧問標準輸出：策略必須可驗證、可執行、可衡量。

## Model-to-Layer Mapping

| 層級 | 心理模型 | 使用方式 | 濫用風險 |
| --- | --- | --- | --- |
| 生理 | Peak-End Rule | 設計高峰體驗與順暢結尾 | 只做表面特效，忽略產品真值 |
| 安全 | Loss Aversion | 用風險逆轉減少損失恐懼 | 以恐嚇話術強迫決策 |
| 安全 | Uncertainty Reduction | 提供透明流程與可預期結果 | 隱藏限制條款 |
| 社交 | Social Proof | 用真實案例與社群背書建立信任 | 偽造數據或評價 |
| 社交 | Commitment & Consistency | 透過小承諾推進長期參與 | 過度綁定造成反感 |
| 尊重 | Signaling Theory | 用等級/徽章提供身份訊號 | 只賣身份，不提供價值 |
| 尊重 | Scarcity (genuine) | 限量策略放大重視程度 | 假限量、假倒數 |
| 自我實現 | Self-Determination | 支持自主、勝任、連結三動機 | 空泛使命，缺乏具體機制 |
| 自我實現 | Identity-Based Motivation | 讓品牌與理想自我對齊 | 情緒綁架、價值勒索 |

## Ethical Guardrails

- 所有心理模型都必須以「幫助使用者更好決策」為前提。
- 禁止以誤導、恐嚇、虛假稀缺或不實社會證明操控決策。
- 每個模型應附上可驗證指標，避免只停留在話術層。
