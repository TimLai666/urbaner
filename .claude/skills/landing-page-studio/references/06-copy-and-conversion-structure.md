# Copy and Conversion Structure

## Required Page Structure

1. Navbar
- 品牌名
- 3-4 個錨點
- 主 CTA

2. Hero
- 一句清楚主張
- 輔助說明
- 主 CTA + 次 CTA（可選）
- 視覺焦點（圖像/SVG/WebGL）

3. Value Props
- 3 張卡片
- 每張包含：標題、一句價值、對應 icon

4. Social Proof
- 指標數字、客戶引用、合作品牌（擇二以上）

5. Final CTA
- 強化行動動機
- 主 CTA 再次出現

6. Footer
- 基本導覽與聯絡資訊

## Conversion Writing Rules

1. 先結果後功能
2. 標題句長盡量短（8-18 字）
3. CTA 使用明確動詞（立即預約、開始試用、下載方案）
4. 每個主區塊只保留一個核心訊息

## Variant Output Rule

當 `variant_mode=batch`：

- 每版保持同一資訊架構
- 僅變動風格、節奏、微互動策略
- 附 `variant_diff_summary`

## Industry Notes (Optional)

- SaaS: 加入 integration/logo belt 與產品畫面
- Hospitality: 強調情境照、感官語彙、預訂路徑
- Finance: 強調信任標記、法遵語氣、風險揭露可讀性
