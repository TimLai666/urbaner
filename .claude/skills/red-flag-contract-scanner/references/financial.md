# 金融契約紅旗清單(Loans, Insurance, Credit)

涵蓋:房貸、信貸、車貸、學貸、信用卡、現金卡、保險(壽險、醫療險、產險、投資型保單)、投資商品(基金、ETF、衍生性金融商品)、P2P 借貸、加密貨幣交易所條款。

金融商品的紅旗特性:
- 有金管會 / 銀行公會 / 保險公會的定型化契約規範,但業者仍會在「應記載事項」邊緣偷塞東西
- 條款多到看不完,業務常用「制式合約大家都這樣簽」帶過
- 牽涉複利、機率、長期 → 紅旗的傷害會被時間放大

## 共通的法定保護

1. **金管會定型化契約應記載及不得記載事項**(各金融商品都有)
2. **消費者保護法** + **金融消費者保護法**(特別針對金融商品)
3. **公平交易法**(對寡占金融機構之約定)
4. **猶豫期 / 契約撤銷權**(保險 10 天、刷卡分期 7 天、部分投資商品有)

## 貸款契約紅旗

### 🚨 紅旗 #1:利率機關槍

**長什麼樣**:
- 「指標利率 + 加碼 [浮動]」 — 指標利率每月變
- 「業者得依市場狀況調整利率」 — 沒上限
- 「逾期利率為原利率 [2-3] 倍」 + 「違約金 [3-10%]」 + 「催收費用」 → 三重收費
- 「分期手續費 [0%]」+ 「手續費 [大額]」+ 「開辦費」 → 真實 APR 超高

**檢查**:
- **APR(年化利率)**:把所有費用 + 利息加總算 APR,跟廣告比
- **利率上限**:浮動利率有沒有 cap?
- **逾期條款**:逾期利率倍率、違約金、催收費三項都看
- **提前還款違約金**:常被忽略的紅旗(下一條詳述)

### 🚨 紅旗 #2:提前清償違約金

**長什麼樣**:
- 「於 [3/5] 年內提前清償,須繳納剩餘本金 [1-3%] 之違約金」
- 「轉貸視為提前清償」
- 「部分提前清償亦適用」

**為何紅旗**:你想換更便宜的貸款 → 被罰錢綁住

**檢查**:違約金是否有期限?是否合理?(實務上半年-1 年內合理,3-5 年偏苛)

### 🚨 紅旗 #3:加速到期條款(Acceleration Clause)

**長什麼樣**:「乙方違反本契約任一條款 / 信用評等下降 / 連續 [N] 期未繳 → 全部本息視為到期應立即清償」

**為何紅旗**:你少繳一期 → 被要求一次還清整筆貸款 → 通常還不出來 → 房子被法拍 / 被告

**檢查**:加速到期的觸發條件是否合理?(連續 3 期未繳是市場慣例,1 期就觸發 → 紅旗)

### 🚨 紅旗 #4:擔保品估價權

**長什麼樣**:「擔保品價值由銀行單方鑑定,如價值不足,借款人應補足擔保 / 償還差額」

**為何紅旗**:房市下跌 → 銀行重估 → 要你補錢 → 補不出來 → 違約 → 加速到期

### 🚨 紅旗 #5:連帶保證人責任無上限

**長什麼樣**:「連帶保證人對借款人之全部債務(包括本金、利息、違約金、訴訟費)負連帶清償責任」

**法律上**:民法第 754 條規定保證人原則上有先訴抗辯權,連帶保證放棄此權利。

**檢查**:擔任連帶保證人是非常重的責任,務必提醒使用者「這條等於你跟借款人一樣要還錢」

## 信用卡 / 現金卡紅旗

### 🚨 循環利率 / 預借現金

- **循環利率上限 15%**(金管會新規)但仍偏高
- 預借現金:手續費 3.5% + 利率 15% + 立即計息(沒有免息期)
- 最低應繳金額陷阱:長期繳最低額 → 利息滾到本金的 2 倍以上

### 🚨 自動分期 / 分期付款條款

- 「消費滿 [N] 元自動轉分期」
- 「分期 0 利率」+ 「實際手續費內含於商品價格」(信用卡公司分潤)
- 「提前清償分期須收取剩餘期數手續費全額」

### 🚨 帳單通知 / 爭議處理

- 「持卡人應於收到帳單後 [N] 日內提出異議,逾期視同確認」
- 「電子帳單視同送達」(若你沒看 email → 沒收到)

## 保險契約紅旗

保險最大的紅旗特性:**理賠不出來**。看保險合約的目標是「假設真的發生事故,我能拿到錢嗎?」

### 🚨 紅旗 #1:除外條款過寬

**長什麼樣**(常見於醫療險、意外險):
- 「既往症不予理賠」(範圍寬到「曾經感冒」也算既往症)
- 「故意行為、犯罪行為、酒駕、自殺不予理賠」(自殺通常 2 年除外,後即理賠)
- 「核子事故、戰爭、暴動不予理賠」(常規)
- 「危險運動(滑雪、潛水、登山)不予理賠」(看細節)
- 「精神疾病、先天疾病、整型手術不予理賠」

**檢查**:把所有「除外」條款列出來,跟你的生活風險對照。

### 🚨 紅旗 #2:理賠定義刁鑽

**長什麼樣**:
- 「重大疾病須符合 [極嚴格] 醫療標準」(例如「中風」須有後遺症滿 6 個月)
- 「住院須住滿 [N] 日方理賠」
- 「實支實付以 [收據正本] 為限,影本不接受」(在台灣大部分人選兩家實支實付,但只能拿一份正本)
- 「失能須達 [極嚴格] 等級」

**檢查**:理賠條件是否合理?台灣保險公會有公定定義,業者私自加嚴 → 紅旗

### 🚨 紅旗 #3:解約金陷阱(投資型 / 還本型保單)

**長什麼樣**:
- 「保單前 [3/5/10] 年解約,僅退還已繳保費 [10/30/50]%」
- 「投資型保單之投資虧損由要保人自行承擔」
- 「附加費用率前 [N] 年較高,逐年遞減」(前幾年的錢都被費用吃掉)

**為何紅旗**:你以為買的是「儲蓄 + 保障」,實際上前幾年解約虧很大,而保障部分常常很弱。

**檢查**:
- IRR(內部報酬率)算出來多少?跟銀行定存 + 純保障保單分開買比較
- 解約費用結構

### 🚨 紅旗 #4:保險公司單方權利

**長什麼樣**:
- 「保險公司得依被保險人健康狀況調整保費 / 拒絕續保」
- 「保險公司得單方修改保單條款,通知後生效」
- 「保險金額得依物價指數調整」(對你不一定有利)

### 🚨 紅旗 #5:健康告知義務

**長什麼樣**:「要保人 / 被保險人應據實告知健康狀況,如有不實告知,保險公司得解除契約」

**陷阱**:
- 業務員常說「沒事啦不用寫」 → 你照做 → 出險時被解約
- 「2 年內就醫紀錄」實務上保險公司用健保資料勾稽,你忘的他都查得到

**檢查**:健康告知是否清楚 + 業務員建議的「省略」風險

## 投資商品紅旗

### 🚨 紅旗 #1:基金 / ETF 費用結構

**長什麼樣**:
- 申購手續費 [1-3%]
- 經理費 / 管理費 [1-2%/年]
- 保管費
- 信託管理費(透過銀行買)
- 信託管理變更費 / 贖回費 / 轉換費

**檢查**:總費用率(TER)是多少?跟同類商品比?

### 🚨 紅旗 #2:衍生性商品 / 結構型商品

**長什麼樣**(雷曼連動債、TRF 等災難案例的條款):
- 「最高保本比例 [非 100]%」
- 「設有 KO / KI 條件,觸及則 [不利後果]」
- 「結算採 [複雜公式]」
- 「商品存續期間不得提前贖回」/ 「提前贖回損失全額」

**為何紅旗**:你看不懂 + 業務員也不一定懂 + 賠錢時你沒得退

**警示**:看到結構型 / 連動債 / 雙元貨幣 / TRF / 累積選擇權 → 強烈建議「不要簽」或找獨立投顧諮詢

### 🚨 紅旗 #3:加密貨幣交易所條款

**長什麼樣**:
- 「平台對被駭、系統故障、智能合約漏洞造成之損失概不負責」
- 「平台得隨時凍結 / 終止帳戶」
- 「資產實際上由平台託管,並非用戶錢包」(custodial wallet)
- 「適用 [境外司法管轄]」(出事你告不到)

**警示**:加密貨幣交易所倒閉 / 跑路是常態(FTX、Mt. Gox 等),條款上的保護幾乎為零。

## 簽金融契約前的實務檢查

1. **算總成本**:把所有費用 + 利息 + 違約風險全部加總
2. **算 APR / IRR**:用真實年化利率比較同類商品
3. **問業務的問題**(寫下來請對方書面回答):
   - 「最壞情況下我會虧多少?」
   - 「想中途解約 / 提前還款的成本?」
   - 「這份合約跟業界其他家比哪裡不一樣?」
4. **冷靜期 / 撤銷權**:
   - 保險 10 天
   - 信用卡分期 7 天
   - 部分投資商品有
5. **諮詢獨立專業人士**:不要只聽業務 / 銀行員(他們有業績壓力),金融商品高金額 / 長期 → 找獨立 CFP / 理財顧問
6. **小心「行員推薦」**:銀行櫃員推薦的商品通常是「行員業績獎金高」而非「適合你」的
7. **簽名前再讀一次小字**:小字 / 附件 / 條款全文,別只看「重點摘要」

## 出事了怎麼辦

如果使用者已經簽了發現問題:

1. **冷靜期內**:無條件解約
2. **已過冷靜期但有不當銷售**(業務員話術誤導、未充分揭露風險):
   - 向銀行 / 保險公司申訴
   - 向金管會申訴(1998)
   - 向金融消費評議中心申請評議(免費,有約束力)
3. **重大瑕疵**:可主張詐欺撤銷 / 錯誤撤銷
4. **長時間爭議**:訴訟前先評議(金融消費評議中心),評議結果若 100 萬以下對業者有約束力
