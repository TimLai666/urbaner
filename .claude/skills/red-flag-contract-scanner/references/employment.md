# 工作 Offer / 聘僱契約紅旗清單(Employment Contract)

涵蓋:正職 Offer Letter、聘僱契約、外包 / 承攬契約、實習契約、派遣合約。

以台灣勞基法為主要依據,跨國公司 / 外商公司聘僱契約常用美國 / 香港 / 新加坡格式,需特別注意這些條款在台灣是否有效。

## 必看的台灣勞動法保護

先檢查下列法定權利是否被剝奪 —— 這些寫了通常無效,但合約還是會寫:

1. **最低工資** + **基本工時**(每日 8 小時、每週 40 小時)
2. **加班費**(平日 1.34 / 1.67 倍,休息日、國定假日另計)
3. **特休**(滿 6 個月 3 天起跳,至滿 10 年 30 天封頂)
4. **資遣費** + **預告期**(勞基法 / 勞退舊新制)
5. **競業禁止合理補償**(勞基法第 9-1 條:不得超過 2 年 + 需合理補償 + 需明確範圍)
6. **就業歧視禁止**(性別、年齡、婚育、宗教等)
7. **加班費換補休須勞工同意**
8. **試用期不得免於勞基法保障**(試用期解僱仍需符合勞基法第 11、12 條事由 + 預告 + 資遣)

## Offer / 聘僱契約最該掃的紅旗

### 🚨 紅旗 #1:競業禁止條款過嚴 / 無補償

**長什麼樣**:
- 「乙方離職後 [2/3/5] 年內,不得從事與甲方相關之任何業務」
- 「不得受僱於甲方之競爭對手 / 客戶 / 合作廠商」
- 「不得自行創業從事相關業務」
- 沒有合理補償(或只寫「於勞基法規定之合理範圍」這種空話)

**勞基法第 9-1 條限制**:
- 期間不得超過 2 年
- 需有合理補償(實務上至少月薪 50%)
- 限制範圍須具體合理(不能寫「相關產業」)
- 雇主有保護必要的營業秘密

**檢查**:
- 期間是否超過 2 年?(超過 = 違法)
- 有沒有具體補償金額?(沒寫 = 違法)
- 範圍是否合理?(寫「全球」「所有相關行業」= 違法)
- 有沒有「離職後仍領補償金」的明確條款?

**協商建議**:要求(1) 期間縮短至 1 年,(2) 補償金至少月薪 50%,(3) 範圍寫具體公司名單而非整個產業,(4) 違反才扣補償金 + 賠償,而非「無條件賠償 X 萬」。

### 🚨 紅旗 #2:智慧財產權無限剝奪

**長什麼樣**:
- 「乙方在職期間之所有發明、創作、著作均歸甲方所有」
- 「包括下班時間、非職務上之創作」
- 「乙方既有作品於入職後授權甲方無限期、無償使用」
- 「未經甲方同意不得對外發表任何文章、演講、開源貢獻」

**法律上**(專利法 / 著作權法):職務上之發明 / 創作 → 公司所有(原則);非職務上 → 員工所有(原則)。但「使用公司資源 + 公司業務範圍內」會變灰色。

**檢查**:
- 是否限縮在「職務上」「業務相關」?
- 你個人既有作品(GitHub repo、部落格、Side project)有沒有排除清單?
- 下班後做 side project / 開源貢獻是否被限制?

**協商建議**:
- 加上「職務上」「使用公司資源或時間」之限定
- 列出個人既有作品清單作為附件,明確排除
- 保留個人開源 / 部落格 / 演講權利,僅約定不洩漏營業秘密

### 🚨 紅旗 #3:獎金 / 紅利追回(Clawback)

**長什麼樣**:
- 「乙方於離職後 [6/12/24] 個月內離職,須返還已領之績效獎金」
- 「股票獎酬於離職時未滿 vesting 期間,全數作廢」
- 「年終獎金於發放日仍在職方可領取 → 12/31 領 → 1/1 離職你拿不到」

**檢查**:
- 哪些獎金有 clawback?(基本工資不能 clawback,但獎金 / 分紅 / 股票可以)
- 追回觸發條件是什麼?(自願離職 vs 被解僱;被解僱事由)
- vesting schedule 多長?
- 公司被併購 / 上市時的處理(accelerated vesting)

**協商建議**:
- 對於不合理 clawback(超過 12 個月)→ 縮短或刪除
- 加上「公司違約 / 重大組織異動」時 vesting 加速條款
- 釐清「自願離職」「被解僱」「合意離職」三種情境分別的處理

### 🚨 紅旗 #4:隨意終止條款(At-Will / 解僱無條件)

**長什麼樣**(外商常見):
- 「Either party may terminate this agreement at any time, with or without cause」
- 「Employment is at-will and may be terminated immediately」
- 「Company reserves the right to terminate employment for any reason」

**法律上**:在台灣,**at-will 條款無效**。台灣勞工受勞基法第 11 條(經濟解僱)、12 條(懲戒解僱)保護,雇主不能隨意終止。

**檢查**:
- 是否寫了 at-will?(外商常見)→ 提醒使用者「在台灣這條無效,但若公司硬要終止你,會用這條當藉口」
- 是否有「適用美國法 / 開曼法」?→ 雙重紅旗
- 試用期條款是否寫「試用期內可隨時終止」?→ 試用期仍受勞基法保障

**協商建議**:即使對方堅持寫 at-will,要求加上「subject to applicable local laws」,並書面確認在台灣適用勞基法。

### 🚨 紅旗 #5:看似豐厚實則無用的福利

**長什麼樣**(看起來很美):
- 「無限休假 unlimited PTO」 → 實務上沒人敢請,而且離職時沒有未休特休折現
- 「彈性工時」 → 實際是「不給加班費」的法律包裝
- 「股票選擇權 [大數字]」 → 看 vesting schedule、看履約價、看 cliff、看離職規定
- 「年薪 [N] 個月 + 績效獎金」 → 績效獎金條件寫得很模糊
- 「責任制 / 無打卡」 → 違反勞基法 + 沒加班費
- 「年終至少 [X] 個月」 → 「至少」可能是 0
- 「享員工分紅」 → 看公司章程,實務上常是 0

**檢查**:
- 把確定能拿到的(底薪)跟有條件能拿到的(獎金、股票、彈性福利)分開
- 年薪計算:寫「12 + 2 個月」是保證 → 寫「依公司營運狀況」 → 不保證
- 股票:vesting period、cliff、履約價、離職怎麼算

### 🚨 紅旗 #6:加班 / 工時條款違法

**長什麼樣**:
- 「乙方為責任制,不適用勞基法工時規定」(只有勞基法第 84-1 條核備過的職類才能責任制,且仍要書面個別約定)
- 「加班需經主管同意方可申請加班費」(實務上主管不會同意 → 等於不給)
- 「同意以補休代替加班費」(原則應勞工選擇,雇主不能強制)
- 「因工作性質特殊,同意每日工時 12 小時 / 每月加班 [超過 46 小時]」(違反勞基法上限)

### 🚨 紅旗 #7:訓練費 / 違約金條款

**長什麼樣**:
- 「乙方須於職務上滿 [2/3/5] 年,未滿者須返還訓練費 [大筆金額]」
- 「公司投入訓練資源 [虛高金額] 元」
- 「於外派 / 培訓期間自願離職須賠償 [數倍年薪]」

**法律上**:
- 公司有提供「具有專業性、額外性」的訓練 → 可約定最低服務年限 + 違約金
- 一般 OJT / 上崗訓練 → 不能要求賠償
- 違約金須合理(成本相當原則)

**檢查**:
- 訓練是不是「對員工帶得走的專業訓練」?(MBA、特定證照)→ 可合理約定
- 還是只是「上手必要的內部訓練」?→ 不能要求賠償
- 違約金是否合理?

### 🚨 紅旗 #8:外派 / 海外工作條款

**長什麼樣**:
- 「派駐期間之適用法律以派駐地為準」
- 「派駐期間之社保 / 退休金由乙方自行處理」
- 「派駐結束後甲方不保證原職務 / 原薪資」
- 「派駐期間如自願返台須賠償」

**檢查**(外派人員額外風險):
- 兩地稅務處理(中華民國綜所稅、派駐地所得稅、雙重課稅)
- 健保 / 勞保 / 退休金提撥是否中斷
- 派駐期滿回任的職務 / 薪資保障
- 配偶 / 子女隨行的支援(機票、學費、住宿)
- 外派加給金額 / 結構

### 🚨 紅旗 #9:保密 / 不挖角條款過寬

**長什麼樣**:
- 「不得洩漏公司任何資訊,包括但不限於 [幾乎所有資訊]」
- 「離職後 [永久 / 5 年] 不得挖角公司員工」
- 「不得挖角 [所有] 客戶 / 供應商 / 合作夥伴」

**檢查**:
- 保密範圍是否限於「營業秘密」(有獨立經濟價值 + 採取合理保密措施 + 非一般知悉)
- 不挖角期間是否合理(1 年內合理)
- 挖角範圍是否限於「主動挖角」 vs 「被動接受聯絡」

### ⚠️ 注意 #1:離職規定

**檢查**:
- 預告期(滿 3 個月 10 天、3 年內 20 天、3 年以上 30 天 → 這是法定下限,可加長)
- 交接義務
- 離職前面談、簽署文件清單
- 離職後保密 / 競業 / 不挖角條款延續多久

### ⚠️ 注意 #2:工作地點 / 職務變更權

**長什麼樣**:「公司得依業務需要調整乙方工作地點 / 職務內容」

**法律上**:雇主調動需符合「五大原則」(基於企業經營必需、不違反勞動契約、對勞工合理工資及其他勞動條件未做不利變更、調動工作地點過遠雇主應提供協助、調動工作合於勞工體能)

**檢查**:條款是否限定「合理範圍 / 通勤合理 / 不降級降薪」?

### ⚠️ 注意 #3:離職後爭議解決

**檢查**:
- 寫「適用 [外國] 法」+ 「[外地] 仲裁」 → 你發生爭議要去外國打官司
- 限制提起訴訟 / 強制仲裁
- 仲裁費用誰付

### ✅ 通常正常的條款

- 標準保密義務(合理範圍)
- 利益衝突揭露義務
- 兼職須事前報備
- 公司財產返還義務
- 標準試用期(3 個月)
- 標準預告期 / 資遣費 / 特休依勞基法

## 簽 Offer 前必須協商的清單

1. **薪資結構**:底薪、獎金、股票、福利明確化
2. **競業條款**:期間、範圍、補償金
3. **IP 條款**:既有作品排除、業餘 side project 保留
4. **離職條款**:預告期、未休特休、未發獎金、股票處理
5. **工作地點 / 職務變更**:合理範圍限定
6. **試用期**:長度、評估標準、未通過的補償
7. **加班 / 責任制**:工時記錄、加班費政策
8. **保密 / 不挖角**:期間、範圍

## 給使用者的提醒

- **Offer 階段是最有議價權的時候**,簽下去就是公司的天下
- 業務 / HR 口頭承諾(「之後可以加薪」「會給股票」「彈性 OK 啦」)→ 全部要求寫進 Offer Letter,寫不進去就是不會發生
- **不要因為怕得罪 HR 不協商**:大家都會協商,協商後 80% 以上 Offer 不會撤回(只有對方覺得你獅子大開口才會撤),且即使撤回也是早點知道公司文化的好事
- 重大爭議(競業 / 股票 / IP)→ 找律師看一次再簽,費用比後續打官司便宜太多
