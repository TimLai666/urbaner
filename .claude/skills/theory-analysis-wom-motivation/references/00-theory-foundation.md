# Theory Foundation

## 理論概念

口碑動機理論（Word-of-Mouth Motivation Theory）用於解釋個體為何主動分享產品或服務經驗。  
分析重點是分享行為背後的心理驅動，而不只是內容本身的正負面。

本 skill 支援跨來源證據分析：

- 訪談摘要
- 客服對話或工單紀要
- 社群貼文與留言
- 研究筆記
- 觀察紀錄

## 主要口碑動機類型

1. 助人利他動機（Altruistic Motivation）
以幫助他人降低搜尋成本與決策風險為目的的分享。

2. 社交認同動機（Social Identity Motivation）
以參與社群互動、取得認同、維持歸屬感為目的的分享。

3. 專業展現動機（Self-Enhancement Motivation）
以展現能力、專業、經驗或成就感為目的的分享。

4. 情緒表達動機（Emotional Expression Motivation）
以情緒宣洩或態度表達為主要目的的分享，常伴隨強烈正負向語氣。

## 研究應用（跨來源證據）

本理論可用於：

- 判讀不同證據來源中，分享行為的主導動機是否一致
- 區分「為了幫助他人」與「為了展現自我」等常見混淆動機
- 評估情緒性訊號是否正在放大品牌風險或傳播擴散
- 產出可追溯的 `theory_annotations`，支援後續策略與溝通設計
