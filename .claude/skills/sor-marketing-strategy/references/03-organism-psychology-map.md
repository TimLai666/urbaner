# Organism Psychology Map

## Purpose

這份檔案用來把「消費者為什麼會動」講清楚。Stimulus 沒有穿過 organism，就不算策略。

## Five Organism Lenses

### 1) Cognitive

要判斷：
- 看不看得懂
- 需不需要想太久
- 能不能快速比較與判斷

常見問題：
- 資訊過多
- CTA 太多
- 價值主張抽象
- 不知道下一步

### 2) Emotional

要判斷：
- 是否產生安心、期待、被理解、好奇、成就感
- 情緒是否支撐目標行為，而不是只創造短暫注意

常見問題：
- 情緒很高，但沒有行動方向
- 太強的 urgency 引發壓迫感

### 3) Trust And Risk

要判斷：
- 品牌是否可信
- 承諾是否可驗證
- 使用成本、退貨風險、時間成本是否被說清楚

常見 trust builders：
- 真實案例
- 可驗證數字
- 清楚條款
- 清楚示範

### 4) Value Appraisal

要判斷：
- 值不值得
- 跟我有沒有關
- 現在做是否比晚點做更合理

常見問題：
- 功能很多，但與使用者情境無關
- 只說便宜，不說適配

### 5) Intrusiveness And Friction

要判斷：
- 有沒有被打擾
- 有沒有被追著跑
- 有沒有被監視感
- 行動門檻是否過高

高風險情境：
- push notification
- 再行銷廣告
- 過度精準的 CRM 文案
- 強制彈窗與高頻 modal

## Interpretation Patterns

### Pattern A: 刺激不足

- 症狀：沒人注意、沒人點、沒人記得
- 常見 organism：沒有被喚起 curiosity/relevance

### Pattern B: 刺激很多，但 response 低

- 症狀：有流量、有點擊、沒轉換
- 常見 organism：trust 不足、risk 高、價值不夠清楚

### Pattern C: 短期有效，長期反感

- 症狀：短期 uplift，退訂、封鎖、反感上升
- 常見 organism：intrusiveness 被低估

## Questions To Ask

- 這個受眾此刻最可能卡在哪個 organism 層？
- 我們在要求他做哪個 response 前，是否先完成對應的心理建設？
- 有沒有一個刺激同時帶來正負雙重 organism？

## Output Reminder

在正式輸出中，organism 不能只寫名詞，至少要寫成：
- `current_state`
- `desired_shift`
- `evidence_or_assumption`
- `what_stimulus_changes_it`
