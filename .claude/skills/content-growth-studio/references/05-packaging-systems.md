# Packaging Systems

## Purpose

Use this reference when the task asks for packaging rather than full content production.

## Title Systems

Use one or more of these title families:

- emotional trigger
- curiosity gap
- direct benefit
- question
- number or list
- contrarian angle
- identity or tribe signal

Good title sets should vary the framing, not just swap adjectives.

## Hook Systems

Common hook patterns:

- pattern interrupt
- painful truth
- surprising claim
- direct audience mirror
- promise of payoff
- open loop

Prefer hooks that match the real content instead of empty hype.

## Thumbnail Concept Rules

When the user wants thumbnail concepts, include:

- main visual subject
- composition direction
- color or contrast idea
- optional text overlay

Keep text overlays short. Usually `2-4` words is enough.

## CTA Systems

Choose the CTA style that fits the asset:

- low-friction click or read-next CTA
- comment CTA
- share or tag CTA
- save or bookmark CTA
- identity CTA
- curiosity CTA

Do not attach an aggressive conversion CTA to content that has not yet earned it.

## Interaction Prompt Systems

Use these when the user wants comment or engagement assets:

- opinion split question
- personal experience prompt
- self-identification prompt
- simple vote or poll
- "tag someone" prompt only when natural

Avoid engagement bait that feels disconnected from the content.

## Screenshot / Share Point Rules

Mark a sentence or mini-block as a screenshot point only when it is:

- self-contained
- memorable in one read
- emotionally or intellectually sharp
- still true outside the surrounding paragraph

Do not overload the draft with too many share points.

## Packaging Priority

When time is limited, prioritize in this order:

1. hook
2. title
3. visual concept
4. CTA
5. interaction layer
