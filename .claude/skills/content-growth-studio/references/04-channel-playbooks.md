# Channel Playbooks

## 1. Article / Newsletter / Long-Form Written Content

Default goals:

- earn the click
- hold the first screen
- maintain section momentum
- create one or more share-worthy lines
- land a clear CTA

Default output pieces:

- title set
- opening paragraph or lead
- section structure with subheads
- screenshot or quote-share points when relevant
- closing CTA

Rules:

- front-load the promise
- keep paragraph rhythm varied
- make transitions carry the reader forward
- use screenshot points sparingly and mark them clearly if the user asked

## 2. Long-Form Video

Common examples include YouTube, but this playbook is not YouTube-only.

Default goals:

- win the first `5-30` seconds
- prevent mid-video drop-off
- create visible segment progression
- place CTA without breaking momentum

Default output pieces:

- hook
- opening setup
- segment-by-segment script
- retention beats
- scene or visual tags
- CTA placement

Rules:

- make the hook concrete fast
- add momentum resets roughly every `60-90` seconds for longer scripts
- mark transitions clearly
- keep one dominant promise per video

## 3. Short-Form Video

Default goals:

- stop the scroll
- make the premise clear in `1-3` seconds
- deliver payoff fast
- keep the ending loopable or comment-worthy

Default output pieces:

- hook line
- beat-by-beat script
- overlay or caption cues when useful
- visual prompt or shot note
- short CTA or engagement prompt

Rules:

- one idea per asset
- compress setup aggressively
- prefer contrast, specificity, and immediacy
- do not bury the payoff

## 4. Social Post / Thread / Engagement Asset

Default goals:

- trigger response, not just passive reading
- create a clear stance, payoff, or relatable mirror
- invite comment, save, share, or click

Default output pieces:

- post or thread copy
- comment prompt
- poll prompt if requested
- share prompt if appropriate
- CTA matched to platform behavior

Rules:

- write for interaction, not only for completeness
- keep the ask low-friction
- use controversy carefully and only when brand-safe
- align the interaction prompt with the content's actual theme

## Cross-Channel Repurposing Rule

When turning one source into multiple channels:

- preserve the same core promise
- rewrite the hook to match channel speed
- shrink or expand proof to fit the format
- change CTA to match the platform's normal behavior
