# 服務創新視角

## 什麼算服務創新

- 用新的想法、技術、流程、組合或服務方式，改善原本的服務價值與交付方式。
- 不一定要是全新市場首創；只要能把原有產業鏈、服務內容、流程或互動方式重組成更有價值的形式，也可以算。

## 常見分類視角

### 依創新程度

- 漸進式
- 模組式
- 架構式
- 突破式

### 依服務對象或內容

- 服務商品創新
- 服務流程創新
- 服務組織創新

### 依企業活動

- 經營管理創新
- 行銷手法創新
- 新商品或品牌開發
- 服務內容創新
- 市場區隔與定位創新
- 多元通路與配送服務創新
- 資源整合創新
- 服務流程創新

## 創新機會來源

### 內部來源

- 意外事件
- 不一致或落差
- 程序需求
- 產業或市場結構變動

### 外部來源

- 人口變數
- 知覺、情緒、意義改變
- 新知識或新技術
- 消費需求與競爭條件變化

## 判斷規則

- 如果題目核心是等待、交接、流程摩擦，偏流程創新。
- 如果題目核心是體驗、內容組合、客製化，偏服務內容創新。
- 如果題目核心是通路、交付方式、接觸模式，偏通路與配送創新。
- 如果題目核心是新客群、新定位、新價值主張，偏市場區隔與定位創新。

## 服務創新流程

可用 Voss 流程整理：

1. 取得市場需求資訊
2. 發展新概念與創意
3. 設計服務雛型
4. 內部測試與執行
5. 透過消費者使用修正
6. 投入市場後持續改善

## 服務創新思維提醒

- 不要把「售後維修」誤認為服務的全部。
- 不要只從技術推力出發，忽略市場拉力。
- 不要把服務理解成無限制討好顧客；應平衡彈性、體驗與合理約束。
- 員工與執行系統很重要，不能只寫對顧客端的表面方案。
