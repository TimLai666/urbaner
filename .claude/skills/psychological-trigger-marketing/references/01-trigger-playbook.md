# Trigger Playbook

這份 playbook 把原始概念整理成可直接套用的心理觸發模組。使用時不要全文搬運，應依目標、受眾與證據量選擇最合適的 2-4 個 trigger。

## 1. 嚇唬 / 失落驅動

- 原始名稱：嚇唬消費者
- 正式副標：失落驅動
- 心理機制：放大「不行動的代價」與「錯過的損失」
- 適用場景：限時促銷、季節性商品、名額制活動、供給真的有限的產品
- 文案做法：
  - 強調錯過成本：`錯過這檔，下一次要再等一年`
  - 強調條件變差：`現在預購有折扣，開賣後恢復原價`
  - 強調時機窗口：`這一波登記截止後，不再加開`
- 風險點：
  - 沒有真實限量、限時依據時，不能硬寫
  - 刺激過強但沒有承接理由，容易引發反感
- 台灣在地示例：
  - 拉拉山水蜜桃採收期短，可寫：`今年採收期更短，錯過這批，再等明年`
  - 招生活動可寫：`早鳥登記今天截止，之後只保留現場候補名額`

## 2. 安撫 / 正當化驅動

- 原始名稱：安撫消費者
- 正式副標：正當化驅動
- 心理機制：幫消費者回答「我為什麼值得買」「我買這個不只是花錢」
- 適用場景：高單價商品、禮盒、選品、地方品牌、支持型消費
- 文案做法：
  - 強調價值與意義：`不只是送禮，更是在支持友善耕作`
  - 強調節省替代成本：`省下自己研究與採買的時間`
  - 強調照顧感：`給家人吃得安心，也讓自己少擔心`
- 風險點：
  - 沒有真實公益、在地連結、友善耕作證據時不能濫用
  - 不能只講價值觀卻不交代產品本身價值
- 台灣在地示例：
  - 在地友善耕作禮盒：`這份禮盒不只體面，也讓你把支持留在土地上`
  - 文創工藝品：`你買到的不只是作品，也是技藝延續的一部分`

## 3. 引誘 / 渴望驅動

- 原始名稱：引誘消費者
- 正式副標：渴望驅動
- 心理機制：先讓人想像擁有後的生活、感官與身份感
- 適用場景：旅宿、餐飲、生活風格、審美消費、體驗型商品
- 文案做法：
  - 用畫面感開場：`在老屋木窗旁醒來，聞到第一杯手沖咖啡的香氣`
  - 用身份投射：`把週末過成你想留下來的樣子`
  - 用感官細節：香氣、觸感、光線、溫度、氛圍
- 風險點：
  - 不能只堆砌漂亮形容詞而沒有具體體驗支撐
  - AI 圖像若過度失真，會破壞信任
- 台灣在地示例：
  - 老屋民宿：`不是住一晚而已，是把一個慢下來的週末真正住進去`
  - 茶品牌：`第一口入口的焙香，會讓你願意把今天慢一點過`

## 4. 共享優惠 / 同伴驅動

- 原始名稱：引誘與激勵、2 人經濟學、嚇唬與稀缺（超商促銷段落）
- 正式副標：同伴驅動
- 心理機制：
  - 社交推進：同伴參與會降低決策阻力，讓「買」變成一起完成的行為
  - 錯失利益焦慮：找不到同伴時，消費者會感到正在失去本可得到的折扣
- 適用場景：雙件價、湊單活動、雙人同行、團購、分享解鎖優惠
- 文案做法：
  - 社交推進句型：`找一位朋友一起買，兩件更划算`
  - 錯失焦慮句型：`錯過雙件價，單買每件就貴更多`
  - 組合訴求：`兩人同行現折`、`雙人方案比單人方案更省`
- 風險點：
  - 若沒有真實價差，不可假造「兩件比較省」
  - 不能把社交壓力寫成羞辱語氣，避免引發反感
- 台灣在地示例：
  - 超商促銷：`買 2 件更划算，揪同事一起湊單`
  - 票券活動：`雙人同行方案，每人平均更省`

## 5. 促發

- 原始名稱：促發（priming）
- 正式副標：心理預熱
- 心理機制：先讓人進入某種感受，再影響後續判斷與行動
- 適用場景：頁首標題、視覺語境、活動頁首屏、報名頁前導文案
- 文案做法：
  - 在第一眼植入關鍵字：`安心、升級、快、稀少、限時、剛好`
  - 用畫面或名詞先定義情境：`品牌經理、限量席次、矽谷實習`
  - 先讓讀者覺得自己正在接近某種更好的版本
- 風險點：
  - 用詞太空泛時，會只剩漂亮話
  - 暗示與後續內容不一致時，會拉高跳出
- 台灣在地示例：
  - 招生活動頁首：`把大學四年，過成你的作品集起點`
  - 課程頁：`從第一堂開始，就用業界節奏要求自己`

## 6. 錨定

- 原始名稱：錨定（anchoring）
- 正式副標：比較基準設計
- 心理機制：先給一個參考點，再讓目標價格或方案看起來更合理
- 適用場景：價格頁、方案頁、套票、報名費、早鳥價、組合購
- 文案做法：
  - 原價與現價並列
  - 先放高價基準，再呈現主推方案
  - 用方案排序控制感知價值
- 風險點：
  - 不得使用虛構原價或不存在的比較對象
  - 若價差邏輯不清，會反而讓人質疑
- 台灣在地示例：
  - 工作坊：`市面同類型課程原價 $12,000，這次校友專案價只要 $800`
  - 雙件組促銷：`單件 299，兩件 498，平均一件再少 50`

## 7. 框架

- 原始名稱：框架（framing）
- 正式副標：決策視角設計
- 心理機制：同樣條件，用損失或收益的說法，會讓判斷偏向不同方向
- 適用場景：報名頁、購物頁、活動宣傳、限購與資格制優惠
- 文案做法：
  - 負面框架：`現在不預購，開學後現貨要多付 20%`
  - 正面框架：`現在預購，直接省下 20%`
  - 資格框架：`僅限完成登記者享有早鳥價`
- 風險點：
  - 誇大損失感會被視為話術
  - 條件若藏太深，容易引起不信任
- 台灣在地示例：
  - 系服預購：`今天登記是 8 折，等現貨就不是這個價格`
  - 活動票券：`先卡位的人，不只是先買到，也先保住優惠資格`

## 使用原則

- 先選主驅動，再選輔助 trigger，不要六種全開
- 高流量曝光優先用 `引誘 + 促發`
- 比較高單價或需要解釋價值的 offer，優先用 `安撫 + 錨定`
- 限時或名額制活動，優先用 `嚇唬 + 框架`
- 若是雙件價、湊單、團購或雙人方案，優先加上 `同伴驅動 + 錨定`
- 若缺乏真實證據，降低 `嚇唬` 權重，改強化 `引誘` 或 `安撫`
