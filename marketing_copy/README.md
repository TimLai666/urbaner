## 行銷框架來源

本文案庫的撰寫參考以下行銷理論框架：

- **心理觸發行銷（Psychological Trigger Marketing）**：稀缺性、損失規避、社會證明、權威背書
- **馬斯洛五層次行銷（Maslow Five Needs Marketing）**：歸屬感 → 自尊 → 自我實現
- **SOR 行銷策略（S-O-R Framework）**：刺激（Stimulus）→ 認知（Organism）→ 反應（Response）
- **產品定位理論（Product Positioning Theory）**：競品錨點定位、價格帶區間
- **購買動機分析（Purchase Motivation Theory）**：功能動機 vs 情感動機
- **口碑動機分析（WOM Motivation Theory）**：分享品牌故事的情感驅力
