# 🇺🇸 V2-US｜「你被騙夠了沒？」

**版本**：V2-US｜男士多功能修剪套組 × 反廉價款踩雷 → 父親節送禮
**平台**：TikTok / IG Reels / YouTube Shorts（美國）
**長度**：60 秒
**對應 Persona**：Marcus（自由設計師，CP 精算 × Reddit 做功課）× Alex（科技業 PM）
**行銷角度**：安全需求（終結「會不會又踩雷」的決策恐懼）→ 收尾上升至歸屬需求（送禮＝表達在乎）
**對應社群組題**：ER-01 × FD-01 × ER-06
**市場依據**：US STP ANOVA Top1 `gift_suitability_men`(F=1299)、Top3 `num_grooming_functions`(F=1193)；US 社群共識「與其買三個廉價款，不如買一個好的」、`#Amazonfail` knockoff fatigue
**旗艦 SKU**：`B0GL2DKVQH`（Beard / Mustache · ★3.78 · 距理想 2.49）
**Hero 商品**：URBANER 7-in-1 Grooming Kit｜USB-C 90 分充滿・120 分續航｜IPX7 全機水洗｜Gift-ready 禮盒（不需額外包裝）

## 腳本正文

---

### 第一幕｜鉤子（0–5 秒）

**畫面**：浴室抽屜「啪」一聲拉開——裡面躺著三四台用過的廉價電剪，不同品牌、不同顏色，刀頭都鈍了、有的還卡著毛。一隻手把它們撥到一邊。

**字幕（白字大字）**：
```
How many did it take
to learn?
```

**音效**：抽屜金屬聲、塑膠互相碰撞，無配樂。

---

### 第二幕｜痛點共鳴（5–18 秒）

**畫面（快切，每格 1.5 秒）**：
1. 手機螢幕：Amazon 五星主圖、閃亮產品 → 切到手上實物，廉價塑膠感，落差明顯
2. 修到一半馬達聲變弱、開始夾毛，男人皺眉閉眼
3. 搜尋「replacement blade」→ 頁面顯示 *Currently unavailable*，整台丟進垃圾桶

**旁白（男聲，低沉平穩，英文）**：
> The photos looked great.
> The motor died in two weeks.
> And the blades? You can't even buy them.

**字幕（螢幕下方）**：
```
The reviews lied. Again.
```

---

### 第三幕｜解法登場（18–38 秒）

**畫面**：
浴室，清晨自然光。男人拿起一台 URBANER 7-in-1，動作乾脆。**同一台機器**換上不同刀頭，依序修 beard line、鬢角、頸線——一鏡到底，不換工具。

**旁白（英文）**：
> One kit. Every detail.
> Built by a factory that's made grooming tools
> for the world's top brands — for 50 years.

**畫面（特寫）**：
- 一條 USB-C 線插上（和手機同一條），充電指示亮起
- 刀頭貼皮膚乾淨滑過，無夾拉
- 修完直接放水龍頭下沖，IPX7 全機水洗，毛屑沖光

**字幕（穿插）**：
```
7-in-1  ·  USB-C  ·  IPX7 washable
One good tool > three cheap ones
```

---

### 第四幕｜情感升格（38–52 秒）

**畫面**：
男人把**同一台機器**放回它的禮盒——盒子本身就是 gift-ready 的質感。鏡頭一轉：父親節，他把這個盒子遞給爸爸。隔天清晨，爸爸自己在鏡子前拿起它，動作自然。

**旁白（英文）**：
> You stopped asking him "do you like it?"
> Because this time, he just uses it.

**字幕（全螢幕，細字）**：
```
The one he'll actually use.
```

**音效**：禮盒蓋輕輕闔上，清晨環境音漸起。

---

### 第五幕｜收尾（52–60 秒）

**畫面**：
URBANER 奧本 logo 淡入，7-in-1 套組靜物圖，白底，極簡，禮盒在側。

**字幕**：
```
URBANER
Details matter.
```

**CTA（右下角小字）**：
```
🔗 Link in bio｜Amazon US · Father's Day Bundle｜Gift-ready, no wrapping
```

**配樂**：全片無配樂，僅環境音與音效。

## 腳本正文

---

### 第一幕｜鉤子（0–5 秒）

**畫面**：清晨，洗面所。逆光，一個男人靠近鏡子端詳自己——指尖沿著下顎鬍線、鬢角邊緣滑過，停在一個極小的「不太齊」的瞬間。

**字幕（白字大字・日文）**：
```
その身だしなみ、
"なんとなく" で選んでいませんか。
```

**音效**：早晨家中低頻環境音、水龍頭遠處滴水，無配樂。

---

### 第二幕｜痛點共鳴（5–18 秒）

**畫面（快切，每格 1.5 秒）**：
1. 包裝印著「超静音」→ 開機，音量明顯偏大，男人皺眉
2. 螢幕顯示「充電 16 時間」→ 男人看錶，閉眼
3. 附件「啪」一聲斷裂，本體還能動，卻只能整台收進抽屜

**旁白（男聲，低沉平穩，日文）**：
> 「静か」と書いてあったのに、うるさい。
> 充電は 16 時間。
> アタッチメントが壊れたら、本体ごと無駄になる。

**字幕（螢幕下方・日文）**：
```
"なんとなく" の道具は、
毎朝あなたを裏切る。
```

---

### 第三幕｜解法登場（18–38 秒）

**畫面**：
洗面所，自然光，桌上放著 URBANER ひげトリマー。男人拿起——動作乾脆，開始修整鬍線、鬢角。

**旁白（日文）**：
> URBANER 奥本。
> スペックを、全部数字で公開する。

**畫面（特寫）**：
- 転盤を回すと「カチッ」と段が決まる——0.5mm 単位、40 段階
- 刃が肌の上をすべる、ほぼ無音、引っかからない
- 使用後そのまま蛇口の下へ——IPX7、本体まるごと丸洗い

**字幕（穿插・数字強調・日文）**：
```
アタッチメント 7個 · 40段階（0.5mm）
騒音 55dB 以下 · IPX7 丸洗い
楽天直営正規品 · 1年保証
```

---

### 第四幕｜情感升格（38–52 秒）

**畫面**：
鏡の前、整った横顔——自惚れではなく「整った」という静かな確信。背広を羽織り、玄関へ向かう後ろ姿。

**旁白（日文）**：
> 身嗜みは、誰かのためじゃない。
> 自分への、基準だ。

**字幕（全螢幕・細字・日文）**：
```
道具で、人は決まる。
```

**音效**：玄関の扉が静かに閉まり、街の音がわずかに立ち上がる。

---

### 第五幕｜收尾（52–60 秒）

**畫面**：
URBANER 奧本 logo 淡入，產品靜物圖，白底，極簡（ひげトリマー＋アタッチメント一式）。

**字幕（日文）**：
```
URBANER 奥本
品味は、細部に宿る。
```

**CTA（右下角小字・日文）**：
```
🔗 プロフィールのリンク｜楽天 URBANER 直営店 · Amazon JP｜正規品・1年保証
```

**配樂**：全片無配樂，僅環境音與音效——「靜」本身就是產品力的展示。
