## 腳本正文

---

### 第一幕｜鉤子（0–5 秒）

**畫面**：清晨，逆光，鏡子前一個男人盯著自己的側臉，鬢角略亂，眼神不滿意。

**字幕（白字大字）**：
```
出門前，你最後看了幾次鏡子？
```

**音效**：城市晨間低頻環境音，無配樂。

---

### 第二幕｜痛點共鳴（5–18 秒）

**畫面（快切，每格 1.5 秒）**：
1. 手拿一台廉價電剪，鬢角修出一個缺口，皺眉
2. 電量格只剩一格，修到一半停機，男人閉眼
3. 理髮廳等候區椅子坐滿，看錶，不耐煩

**旁白（男聲，低沉平穩）**：
> 夾毛、沒電、等預約……
> 你不是要求太高——
> 你只是一直在用錯的工具。

**字幕（螢幕下方）**：
```
你浪費的，不只是時間。
```

---

### 第三幕｜解法登場（18–38 秒）

**畫面**：
浴室，清晨自然光，桌上放著 URBANER 修剪器。
男人拿起來，動作乾脆，開始修整鬢角、beard line。

**旁白**：
> URBANER 奧本——
> 精密間距刀頭，0.5mm 到 19mm，
> 設定長度，每次結果一樣。

**畫面（特寫）**：
- 刀頭貼近皮膚，乾淨俐落滑過，無夾拉
- 轉盤微調特寫，喀一聲定位
- 沖水清潔，毛屑全沖走，5 秒完成

**字幕（穿插）**：
```
不夾毛 · 恆功率馬達 · 防水好清潔
```

---

### 第四幕｜情感升格（38–52 秒）

**畫面**：
男人放下修剪器，對鏡子看了一眼——
不是自戀的那種，是「準備好了」的那種。
拿起外套，走向門口，背影。

**旁白**：
> 你對工具的要求，
> 和你對自己的要求——
> 其實是同一件事。

**字幕（全螢幕，細字）**：
```
Details matter.
```

**音效**：門關上，城市聲浪漸起。

---

### 第五幕｜收尾（52–60 秒）

**畫面**：
URBANER 奧本 logo 淡入，產品靜物圖，白底，極簡。

**字幕**：
```
URBANER 奧本
品味源自細節
```

**CTA（右下角小字）**：
```
🔗 連結在 bio｜美國 / 日本 Amazon 有售
```

**配樂**：全片無配樂，只有環境音與音效——安靜感本身就是產品力的展示。
