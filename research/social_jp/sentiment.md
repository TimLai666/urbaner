## テーマ A：ヒゲ・髭 — 「青ヒゲ」と「清潔感」の二項對立

### コア感情
- **「青ヒゲ」コンプレックス**：濃い髭が目立つことに対する深い不安
- **「ヒゲがある男性」vs「ない男性」の女性評価で 約85% が「ない方が好き」**
- 「ヒゲを伸ばすのは自分に自信が無いというコンプレックスの打消し」という心理学的解釈
- 一方で**おしゃれな無精ヒゲ・デザインヒゲ**は「漢らしさ」「清潔感」両立で評価される

### 主要な痛點
1. **青ヒゲ**：剃ってもうっすら見える
2. **朝剃って夕方には伸びる**（深剃り限界）
3. **カミソリ負け / 肌荒れ**
4. **濃さに自信がない / 整え方がわからない**
5. **不均一な濃淡**（口下、頬、首）
6. **ハサミだと長さがバラバラ**

### ねがい / Wish List
- ✨ **0.5mm 単位の細かい長さ調整**
- ✨ **AI 髭濃さ自動制御**（ラムダッシュ採用）
- ✨ **静音モーター**（家族・パートナーに迷惑かけない）
- ✨ **防水・丸洗い可**
- ✨ **ヒゲ吸引**（部屋汚さず）
- ✨ **自動研磨**（切れ味維持）
- ✨ **専門家・脱毛クリニック推薦の信頼性**

### ギフトシーン
- **父の日**：シェーバー / ヒゲトリマーは「父の日.jp」ランキングで定番
- 「お父さんの愛用シェーバーと同じタイプでワンランク上」推奨戦略
- パナソニック・ブラウン・フィリップス・マクセルイズミ「4社なら品質間違いなし」
- **クリスマス・誕生日**：彼氏・夫にも
- ヒゲを生やしているお父さん・彼氏には「ヒゲトリマー」推奨
- 父の日.jp は「頑張るお父さんに新しい髭剃り体験」訴求

### 文化情境
- **「整え方」の儀式性**：くしで髭をとかしながら → ハサミ/トリマーで丁寧に
- **イケメン製造論**：プロが伝授する正しいヒゲの整え方
- **ヒゲ脱毛市場の急成長**：根本解決を求める層
- **「無精髭デザイン」 vs 「クラシックスタイル」**：自分らしさの選択

## テーマ B：ボディトリマー / VIO — 「衛生」と「恥ずかしさ」の心理戰

### コア感情
- **VIO 部位への躊躇い**：「恥ずかしくて言い出しづらい」が知恵袋常出題
- **彼氏／彼女の反応への不安**：「ハイジニーナに引かれない？」
- **「処理してると気付かれたくない」**：自然なデザイン需求
- 「衛生面・清潔感・ムレ軽減」が動機 Top3

### 主要な痛點
1. **チクチク感・ヒリヒリ感**
2. **剃り残しで時間がかかる**
3. **見えない部分の不安**（鏡が必要）
4. **「肌に直接刃が触れない」設計の必要性**
5. **VIO は痛みリスクが高い**

### ねがい / Wish List
- ✨ **アタッチメントで刃が肌に直接触れない**
- ✨ **VIO 適用安全設計**
- ✨ **浴室で使える**（防水）
- ✨ **3,000円〜10,000円**価格帯
- ✨ **肌当たりがやさしい**（パナソニック ER-GK23-A モニター結果）

### 文化情境
- **「メンズ脱毛」サロン市場成長**：ヒゲ脱毛ついでに VIO
- **アンダーヘアトリマー**は男女両用化進行
- **パナソニック ER-GK 系列がほぼ独占**（ER-GK60-W から GK82-K まで）
- **温泉・銭湯文化**：日本特有の「他人の目」プレッシャー
- **ジム・スポーツ**：清潔感の延長として

## テーマ C：鼻毛・耳毛 — 「身嗜み」のミニマル基本

### コア感情
- **「マナーとして」**：日本では身嗜みの一部として浸透
- **電動 vs 手動の選択論争**：知恵袋常設質問
- **「痛くない」が絶対条件**

### 主要な痛點
1. **痛い経験のトラウマ**：拔いた経験者多数
2. **電動でも切れ味落ちると引っ張る**
3. **電池式の不便さ**
4. **部分耐水問題**：丸洗い不可で衛生不安
5. **円筒型と剃刀型どちらが良いか分からない**

### ねがい
- ✨ **「ほとんど痛くない・あまり痛くない」が大半に**
- ✨ **本体丸洗い可能**
- ✨ **IPX7 防水**
- ✨ **毛くず吸引機能**
- ✨ **円筒型 = 鼻の楕円形にフィット**
- ✨ **手動式 = 永久故障無し（皇家シルバーブレット派の根強さ）**

### 競品文化
- **Panasonic ER-GN71-K / ER-GN32-K**：定番中の定番
- **Maxell イズミ IZN-C240-K**：マイベスト#1
- **Hitachi**：手頃価格＋IPX7
- **皇家 ET-3 (Royal Silver Bullet)**：手動派の隠れ名作
- **「パナ・日立・フィリップス」三強**

### 文化情境
- **「身嗜み」概念**：日本特有の自分管理文化
- **マナーカッター**：男性ビジネスマンの基本装備認知
- **エチケットカッター**：女性向けの言い方
- **おしゃれ度ではなく「不快感を与えない」基本ライン**

### コア感情
- **「カミソリ負け全然しなくなった」**（フェリエユーザー実感）
- **女性の朝の儀式の一部**：化粧前の産毛・眉毛処理

### 主要な痛點
1. **カミソリ肌荒れ**
2. **眉毛 vs 顔の処理の使い分け**
3. **女性向け＝可愛さ求める**

### ねがい
- ✨ **眉カバー＋コーム付き**（4mm まで保護して剃れる）
- ✨ **税込2,000円前後**コスパ
- ✨ **フェリエ ピンク**：パナソニックの定番
- ✨ **医療脱毛クリニック指定**信頼感

## テーマ E：電気シェーバー / メンズシェーバー — 「朝の戦い」

### コア感情
- **朝の限られた時間との戦い**：時間短縮 = QOL
- **敏感肌でも使えるか**の不安
- **長年使い続ける愛着**：「3-4年愛用してたが壊れたので新調」
- **ブラウン vs ラムダッシュ**は永遠の論争（知恵袋・価格.com 常設）

### 主要な痛點
1. **深剃り限界**：朝→夕方に少し生える
2. **肌荒れ・ヒリヒリ感**
3. **充電時間が長い**
4. **価格が高い**：上位機種は2-5万円
5. **替刃が高い**（特にブラウン Clean&Renew カートリッジ）

### ねがい
- ✨ **AI 髭濃さ自動制御**
- ✨ **5層・6層刃で深剃り**
- ✨ **肌への引っ掛かり無し**
- ✨ **時間短縮**
- ✨ **静音**

### ギフト
- **父の日 #1 商品ジャンル**：シェーバー
- **パナソニック・ブラウン・フィリップス・マクセルイズミ**
- 価格帯：1万円〜5万円

### 文化情境
- **価格.com 比較文化**：詳細スペック比較が購買中心
- **「ラムダッシュ PRO」「Series 9 Pro」**は最上位機種ブランド戦争
- **理容師・barber 寄り**：BaBylissPRO FX3 はプロ用市場
- **AI 機能・スマート化**は富裕層向け差別化

## テーマ F：ペットバリカン / ペットトリミング — 「家族のケア」

### コア感情
- **節約モチベーション**：トリミング代 5,000-10,000円/中小型犬、1万-1.6万円/大型犬。月1で年間 6-19 万円
- **「人間の美容院代より愛犬のトリミング代の方が高くつく」**（知恵袋ユーザー声）
- **罪悪感**：愛犬がストレス、でも放っておけない
- **失敗への恐怖**：バリカン負け、皮膚を傷つける

### 主要な痛點
1. **愛犬がストレスでフケ・荒れ・バリカン負け**
2. **動作音が大きいと音敏感な子は震える**
3. **連続使用で発熱**
4. **アタッチメントとバリカンの間に毛が詰まる**
5. **小型犬の肉球の間が刈りにくい**
6. **素人カットで仕上がりが心配**

### ねがい
- ✨ **静音設計**（音に敏感な子向け）
- ✨ **切れ味抜群**＝引っ掛からない
- ✨ **軽量で長時間疲れない**
- ✨ **コードレス充電式**
- ✨ **12,000Pa 強力吸引一体型**
- ✨ **トリマー監修・プロ推奨ブランディング**

### 解決行動
- **「皮膚に平行に刃を当て、毛の流れに沿って動かす」**
- **力を入れず軽く持つ**
- **嫌がる場合は「1日1本」「1週間に1本」**
- **無理せずサロン・動物病院へ**
- **通信教育でトリマー学習**派も増加

### 競品文化
- **Pateker（B01K73KIMO）**：「20年プロトリマー推薦」キャッチで日本市場長期No.1
- **Looffy**：鋭さ評価
- **oneisall 4-in-1**：海外勢代表
- **Panasonic ペット用**：純血パナ派

### 文化情境
- **「毛のお家族化」**：愛犬・愛猫が家族の一員
- **トリミングサロンの料金透明化**：くらしのマーケット、Coo&RIKU など予約サイト
- **トリマー資格化**：通信教育で素人参入増加
- **コロナ後の自宅化**：トリマー予約取りづらく → 自宅 DIY 拡大

## テーマ G：ペット爪切り / ネイルグラインダー — 「血を見たくない」

### コア感情
- **「黒爪は血管が見えない＝深爪リスク」**：飼主の最大不安
- **失敗時の出血恐怖**：止血粉（クイックストップ）需要
- **電動派 vs 手動派**：使い分け論

### 主要な痛點
1. **黒爪の血管位置不明**
2. **怖がって暴れる**
3. **切りっぱなしでザラザラ角が残る**
4. **出血が止まらない時の対処不明**

### ねがい
- ✨ **安全ガード付き**（切れすぎ防止）
- ✨ **双刃 plier 式**（爪を両面から）
- ✨ **ステンレス・合金鋼の永久利用感**
- ✨ **「断面が湿った質感になったらストップ」教育**
- ✨ **静音 50dB 程度**
- ✨ **LED ライト＋血管位置確認**
- ✨ **2 速・3 ポート選択**
- ✨ **1時間充電で3-4 時間稼働**

### 解決行動
- **日光・強光で透かして血管位置確認**
- **「クリッパーで粗→グラインダーで仕上げ」コンボ推奨**
- **電動ヤスリは「押し当てるだけ」音が苦手な犬にも**

### 競品
- **HAWATOUR（B07RSM1NG6）**：Amazon JP No.1（9,964 レビュー）安全ガード派
- **Safari 770045**：プロ用伝統派
- **猫壱**：日本製合金鋼
- **VIWIK**：超静音電動の日本ヒット
- **Boshel**：海外勢

## テーマ H：ベビーバリカン — 「初めての我が子の散髪」

### コア感情
- **記念日的儀式**：「初めての散髪」は思い出
- **緊張・不安**：怪我させたくない
- **泣かれたくない**：トラウマ化避けたい
- **美容院デビュー躊躇い**：「赤ちゃんが緊張する慣れない場所」
- **節約**：自宅カット 28% vs 美容院 28% vs 家族カット 71%

### 主要な痛點
1. **赤ちゃんが大きな音で泣く**
2. **動き回ってカットしにくい**
3. **「静音」表記でも実は振動音**
4. **吸塵機能弱い → 髪の毛だらけ**
5. **「散髪嫌いになる」恐怖**

### ねがい
- ✨ **38dB（ささやき声程度）の超静音**（エジソンママ「静音バリカンプロ技」）
- ✨ **50dB 以下は理想**
- ✨ **60dB 以下は最低ライン**
- ✨ **防水水洗い**
- ✨ **8 種類アタッチメント**
- ✨ **0.3mm 単位＋3 段速度**
- ✨ **コードレス**

### 解決行動
- **寝ているとき切る**（71%親が選択）
- **複数回に分けて手早く**
- **動画・テレビで気を引く**
- **「trim」「style」言葉使い分け**（haircut 避ける）
- **家族カット 71% > プロ 28%**

### 文化情境
- **「初めて」記念**：写真撮って家族 LINE 共有
- **筆記念品**：胎毛筆（産毛で作る筆）伝統
- **赤ちゃんの個人差**：タイミングは生後半年〜1歳が大半
